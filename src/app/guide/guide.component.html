Description

Esta herramienta esta pensada unica y exclusivamente para la generacion de modelos de entidades con un esquema espesifico para su uso en la aplicacion Eva dessarrollada para generar CRUD Operation en aplicaciones de Angular.

Atajos de comandos

* Si se ejecuta el comando dentro en la arpeta raiz de un proyecto de angular puede presindir de la ocion -o para el output directory

* Si se ejecuta el comando dentro en la arpeta raiz de un proyecto de angular y se tiene un archivo modelo de entidad en la raiz del proyecto con el nombre model puede presindir de la ocion -m para el model

* Tarea Perfecionar los mensages de la ayuda del cli con estos atajos
